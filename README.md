# StoreInventory

Console store inventory system written in C++ using inheritance and polymorphism.

## Features

- Add different product types (Food, Electronics, Clothing)
- Each type has unique properties and info display
- View all products with type-specific details
- Polymorphic storage in a single collection

## Product types

- Food: name, price, expiry days
- Electronics: name, price, warranty months
- Clothing: name, price, size

## Technologies

- C++
- OOP: inheritance, polymorphism, virtual methods, override
- Abstract base class (pure virtual functions)
- Pointers and dynamic memory
- STL (vector)

## How to run

1. Clone the repository
2. Open `StoreInventory.cpp` in Visual Studio or any C++ compiler
3. Build and run
